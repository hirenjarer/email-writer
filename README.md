# Email Writer Tool
A web-based tool that helps users generate email content using AI.

### Features
- Generate emails using AI
- Easy-to-use UI
- Fast backend API

### Tech Stack
- Frontend: React
- Backend: Spring Boot & Spring AI 
- API: REST & GEMINI API

### Project Structure
- frontend → UI (React)
- backend → Spring Boot API

### How to Run
#### Backend
```bash
mvn spring-boot:run
```

#### Frontend
```bash
npm install
npm start
```
